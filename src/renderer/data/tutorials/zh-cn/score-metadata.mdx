# 乐谱元数据

本页列出了所有影响整体歌曲的元数据标签。

建议在文件最开始指定这些标签。

## `\\title`

```markdown
\title (value template textAlign)
```

**描述：** 设置歌曲标题。

**参数：**

| 名称 | 描述 | 类型 | 必需 |
|------|-------------|------|----------|
|  `value` | 要设置的值 | `String` | 是  |
|  `template` | 用于渲染文本的模板 | `String` | 否  |
|  `textAlign` | 乐谱上文本的对齐方式 | `left\|center\|right` | 否  |

**参数值：**

| 名称 | 值 |
|------|--------|
| `textAlign` | left — 左对齐文本；center — 居中对齐文本；right — 右对齐文本 |

**示例：**

```alphatex
\title ("Song Title" "Title: %TITLE%" left)
C4
```
