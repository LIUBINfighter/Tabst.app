# 音符属性

在音符级别定义的属性会在乐谱中显示各种效果和注释。历史上，这些属性被大量缩写以减少应用它们时需要编写的文本。

将来，alphaTab 可能会改为更详细但更清晰的替代方案。

## `nh`

```markdown
nh ()
```

**描述：** 对音符应用自然泛音效果（用于品乐器）。

**示例：**

```alphatex
:8 3.3{nh} 3.3{ah} 3.3{ph} 3.3{th} 3.3{sh}
```

## `ah`

```markdown
ah value
```

**描述：** 对音符应用人工泛音效果（用于品乐器）。

**参数：**

| 名称 | 描述 | 类型 | 必需 |
|------|-------------|------|----------|
|  `value` | 泛音值决定了演奏时改变音符音高的相对品偏移。有关详细映射，请参阅上游文档。 | `Number` | 否  |

**示例：**

```alphatex
C4 {volume 8} D4 E4 {tempo 16} F4
```

```markdown

```

**描述：** 为音符添加轻微颤音效果。

**示例：**

```alphatex

```


## `sl`

```markdown
sl ()
```

**描述：** 为音符添加连音滑弦效果。

**示例：**

```alphatex
3.3{sl} 4.3 3.3{ss} 4.3 |
3.3{sib} 3.3{sia} 3.3{sou} 3.3{sod} |
3.3{sib sou} 3.3{sib sod} 3.3{sia sod} 3.3{sia sou} |
x.3{psd} 3.3 |
x.3{psu} 3.3 |
```

## `ss`

```markdown
ss ()
```

**描述：** 为音符添加移位滑弦效果。
