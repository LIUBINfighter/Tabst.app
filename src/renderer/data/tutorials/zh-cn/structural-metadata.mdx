# 结构元数据

对于某些歌曲，你可能需要定义歌曲具有多个音轨、谱表或声部。

* 多音轨用于一首歌曲中多个乐器一起演奏时。
* 多谱表通常用于编写大谱表钢琴记谱（左手/右手）或
  为可读性拆分声部。
* 多声部通常用于编写多个独立节拍和音高同时演奏的记谱。
  这可以是多个声乐一起演唱（因此称为声部），或特殊的演奏技巧（在同一乐器上演奏基础节奏和旋律）。

在 AlphaTex 中，新的音轨、谱表和声部通过其各自的元数据标签在小节开头指定：

* `\track`
* `\staff`
* `\voice`

alphaTab 将这类元数据标签称为"结构"标签，因为它们定义了歌曲的一般结构。

初始的音轨、谱表或声部是隐式启动的。只有在"第二个"项目上你才需要指定标签。为了清晰起见，
建议如果有多个音轨、谱表或声部，也指定初始项目。

## `\\track`

```markdown
\track (name shortName)
```

**描述：** 开始一个新音轨

**参数：**

| 名称 | 描述 | 类型 | 必需 |
|------|-------------|------|----------|
|  `name` | 音轨的全名 | `String` | 否  |
|  `shortName` | 音轨的短名 | `String` | 否 `/*从 name 计算得出*/` |

**示例：**

```alphatex
\track "First Track" "frst"
    C4 D4 E4 F4
\track
    C5 D5 E5 F5
```

**属性：**

### `color`

```markdown
color colorCode
```

**描述：** 数据模型保存有关音轨颜色的信息，用户界面可能使用这些信息来视觉上区分它们。这些数据主要源自 Guitar Pro 文件格式，其中颜色用于在音轨选择器和一些其他可视化中区分音轨。它不会影响乐谱的颜色，但可以在自定义 UI 中使用。

**参数：**

| 名称 | 描述 | 类型 | 必需 |
|------|-------------|------|----------|
|  `colorCode` | 任何支持的颜色格式的颜色代码 | `String` | 是  |

**示例：**

```alphatex
\track { color "#FF0000" }
1.1 2.1 3.1 4.1
```

### `systemsLayout`

```markdown
systemsLayout (numberOfBars)
```

**描述：** 定义每行显示的小节数。

`systemsLayout` 和 `defaultSystemsLayout` 允许配置系统布局。系统布局定义了如果通过 `systemsLayoutMode` 设置启用，每行（行）应显示多少小节。

**参数：**

| 名称 | 描述 | 类型 | 必需 |
|------|-------------|------|----------|
|  `numberOfBars` | 为每行（行）定义它应包含的小节数 | `Number[]` | 否  |

**示例：**

```alphatex
\title "Single Track"
\track { systemsLayout (2 3 2) }
:1 c4 | c4 | c4 | c4 | c4 | c4 | c4
```

### `defaultSystemsLayout`

```markdown
defaultSystemsLayout numberOfBars
```

**描述：** 定义每行显示的默认小节数。

**参数：**

| 名称 | 描述 | 类型 | 必需 |
|------|-------------|------|----------|
|  `numberOfBars` | 为每行（行）定义它应包含的小节数 | `Number` | 是  |

**示例：**

```alphatex
\title "Single Track"
\track { defaultSystemsLayout 2 }
    :1 c4 | c4 | c4 | c4 | c4 | c4 | c4
```

### `solo`

```markdown
solo ()
```

**描述：** 将音轨设置为独奏播放。

**示例：**

```alphatex
\track { solo }
1.1 2.1 3.1 4.1
\track 
10.1 11.1 12.1 13.1
```

### `mute`

```markdown
mute ()
```

**描述：** 将音轨设置为静音。

**示例：**

```alphatex
\track 
1.1 2.1 3.1 4.1
\track { mute }
10.1 11.1 12.1 13.1
```

### `volume`

```markdown
volume value
```

**描述：** 设置音轨音量。

**参数：**

| 名称 | 描述 | 类型 | 必需 |
|------|-------------|------|----------|
|  `value` | 要设置的音量（0-16） | `Number` | 是  |

**示例：**

```alphatex
\track { volume 0 }
1.1 2.1 3.1 4.1
\track { volume 16 }
10.1 11.1 12.1 13.1
```

### `balance`

```markdown
balance value
```

**描述：** 设置音轨平衡。

**参数：**

| 名称 | 描述 | 类型 | 必需 |
|------|-------------|------|----------|
|  `value` | 要设置的平衡（0-16；8 为居中） | `Number` | 是  |

**示例：**

```alphatex
\track { volume 0 }
1.1 2.1 3.1 4.1
\track { volume 16 }
10.1 11.1 12.1 13.1
```

### `instrument`

```markdown
// [1]: 将 MIDI 乐器设置为 MIDI 程序编号
instrument program
// [2]: 将 MIDI 乐器设置为 MIDI 程序名称
instrument programName
// [3]: 将乐器设置为打击乐
instrument percussion
```
