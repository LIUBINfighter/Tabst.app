# 节拍属性

在节拍级别定义的属性会在乐谱中显示各种效果和注释。历史上，这些属性被大量缩写以减少应用它们时需要编写的文本。

将来，alphaTab 可能会改为更详细但更清晰的替代方案。

## `f`

```markdown
f ()
```

**描述：** 为节拍添加淡入效果。适用于所有音符。

**示例：**

```alphatex
(0.1 2.2 2.3 2.4 0.5).2 { f }
```

## `ds`

```markdown
ds ()
```

**描述：** 将节拍标记为死击节拍。

由于此类节拍上不应该有音符，只需使用 () 表示空节拍，它不是休止符。


**示例：**

```alphatex
().4 {ds} ().4 {ds}
```

## `tu`

```markdown
// [1]: 
tu simple
// [2]: 
tu (numerator denominator)
```

**描述：** 为节拍时长添加连音。

有一些内置的简单连音变体，或者你可以指定精确的划分。

**参数：**

| 重载 | 名称 | 描述 | 类型 | 必需 |
|----------|------|-------------|------|----------|
| `[1]` |  `simple` | 简单连音 | `Number` | 是  |
| `[2]` |  `numerator` | 连音分子 | `Number` | 是  |
| `[2]` |  `denominator` | 连音分母 | `Number` | 是  |

**参数值：**

| 重载 | 名称 | 值 |
|----------|------|--------|
| `[1]` | `simple` | 3 — 3:2 连音；5 — 5:4 连音；6 — 6:4 连音；7 — 7:4 连音；9 — 9:8 连音；10 — 10:8 连音；12 — 12:8 连音 |

**示例：**

```alphatex
C4 {tu 3} * 3 |
C4 {tu 5} * 5 |
C4 {tu 5 2} * 5
```

## `txt`

```markdown
txt text
```

**描述：** 为节拍添加文本注释。

**参数：**

| 名称 | 描述 | 类型 | 必需 |
|------|-------------|------|----------|
|  `text` | 显示在节拍上方的文本 | `String` | 是  |

**示例：**

```alphatex
C4 {txt "This is a C4 Note"}
```

## `lyrics`

```markdown
// [1]: 
lyrics text
// [2]: 
lyrics (line text)
```

**描述：** 为节拍添加歌词文本。

这是将确切歌词应用到节拍的替代方式，而不是使用 `\lyrics` 元数据，后者使用特殊语法将文本分布到节拍上。

**参数：**

| 重载 | 名称 | 描述 | 类型 | 必需 |
|----------|------|-------------|------|----------|
| `[1]` |  `text` | 要显示的歌词 | `String` | 是  |
| `[2]` |  `line` | 行号（例如用于多段歌词） | `Number` | 是  |
| `[2]` |  `text` | 要显示的歌词 | `String` | 是  |

**示例：**

```alphatex
C4 {txt "This is a C4 Note"}
```

## `timer`

```markdown
timer ()
```

**描述：** 为节拍添加时间戳标记。

计时器显示第一次演奏节拍时的确切时间戳（考虑反复等方面）。


**示例：**

```alphatex
\tempo 120
3.3.4 { timer } 3.3.4*3 |
\ro 3.3.4 { timer } 3.3.4*3 |
3.3.4 { timer } 3.3.4*3 |
\jump fine 3.3.4 { timer } 3.3.4*3 |
\ae (1) 3.3.4 { timer } 3.3.4*3 |
\ae (2 3) \rc 3 3.3.4 { timer } 3.3.4*3 |
3.3.4 { timer } 3.3.4*3 |
\jump DaCapoAlFine 3.3.4 { timer } 3.3.4*3 |
3.3.4 { timer } 3.3.4*3
```
