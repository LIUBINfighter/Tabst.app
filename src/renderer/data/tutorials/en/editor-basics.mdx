# Editor Basics

The editor supports day-to-day text editing for AlphaTex, with fast feedback from the preview pane.

## Editing Features

- Basic text editing
- Auto-save behavior
- Syntax highlighting
- Lightweight formatting consistency from project tooling

## Working with Preview

- Keep the editor and preview side by side.
- Make small edits and verify rendering continuously.
- Use playback controls to validate rhythm and timing.

## Shortcuts

- `Ctrl+Z` / `Cmd+Z`: Undo
- `Ctrl+Y` / `Cmd+Y`: Redo
- `Ctrl+S` / `Cmd+S`: Save
- `Ctrl/Cmd+O`: Open file switcher
- `Ctrl/Cmd+Shift+P`: Open global command palette

## Practical Advice

- Keep directives near the section they control.
- Split long ideas into smaller bars for easier debugging.
- Prefer explicit structure over dense one-liners when collaborating.
