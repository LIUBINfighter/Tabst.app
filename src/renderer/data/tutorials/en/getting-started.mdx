# Getting Started

Welcome to Tabst! This guide helps you go from opening the app to writing and playing your first score.

## Basics

Tabst is a guitar tab editor built around AlphaTex, a plain-text notation format. You can edit text on the left and see rendered notation on the right.

## Typical Workflow

1. Create or open a `.atex` file.
2. Enter AlphaTex content in the editor.
3. Preview score rendering in real time.
4. Adjust playback settings and press Play.
5. Iterate quickly with text-first edits.

## Example

```alphatex
\title "Sample Track"
\time 4/4
\key C
.
:4 0.3 2.3 3.3 5.3 |
```

## Tips

- Start from small phrases and build measure by measure.
- Use metadata to keep your files searchable.
- Keep files plain and portable for easier sharing and versioning.
