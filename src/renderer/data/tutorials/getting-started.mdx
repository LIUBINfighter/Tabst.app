# 快速开始

欢迎使用 Tabst！这是一个示例教程页面。这里可以放入多步引导、视频或示例文档。

## 基础操作

Tabst 是一个用于编写吉他谱的编辑器，使用类似 Markdown 的纯文本格式。

## MDX 组件测试

这个教程使用 MDX 格式，可以直接在 Markdown 中使用 React 组件。下面是一些测试组件：

</TestButton>
